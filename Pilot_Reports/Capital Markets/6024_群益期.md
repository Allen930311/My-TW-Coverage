# 6024_群益期

## 業務簡介
**板塊:** Financial Services
**產業:** Capital Markets
**市值:** 13,850 百萬台幣
**企業價值:** 4,626 百萬台幣

群益期貨股份有限公司 (6024) 是台灣領先的期貨與衍生性金融產品經紀商之一。業務涵蓋國內外期貨及選擇權經紀、自營、期顧、期經與槓桿交易商業務。群益期貨以強大的資訊技術(IT)研發能力著稱，建置了專為高頻交易與法人客戶量身打造的全球直接下單系統。除了深耕台灣市場，公司也積極拓展香港等海外據點，放眼大中華區與國際機構法人業務。

## 供應鏈位置
**上游 (各大國際交易所與清算所):**
- **跨板塊報價源與結算端:** 費心取得規模覆蓋美歐亞板塊之全球十二家指標型期貨交易所 (包含 [[CME]]、[[EUREX]] 等) 之直接全面結算會員或交易會員資格，並直連清算結算機構。

**中游 (國際化期貨經紀樞紐):**
- **海期經紀與法人通道:** **群益期貨 (6024)** (為群益證券子公司更為台灣前兩大頂流海外期貨經紀商，強力主導並自建專屬於高端法人與極致高頻客群的 DMA 全球直接下單系統，跨市場據點廣布台灣與香港等金融中心)。

**下游 (全球期權避險與跨洋結算):**
- **衍生金融對手方:** 遍佈大中華區並強烈尋求海外跨區跨市場時差避險或投機價差套利操作之全球中大型法人機構與高頻演算法交易散戶。

## 主要客戶及供應商
### 主要客戶
- **海期外資法人與量化自營群:** 高度需求極度低延遲與極致併發效能下單系統，以操作海標期短單 (如期金/期油等) 規格之跨國主力外資機構、造市商自營交易室、與撰寫 API 下單之高端工程師量化群散戶。
- **國內期權經紀與訂閱顧問用戶:** 委託其旗下「群益期顧」深度進行程式機器人交易實戰輔導，或交由「期經」直接全權專屬委託代操之高資產信託投資人，及一般進行常態型台指期、選擇權日盤/夜盤交易散戶大眾。

### 主要供應商
- **頂級資訊機房伺服器與專網系統:** 由於「獨立 IT 軟體研發」與「極速通訊線路」為其絕對競爭力命脈，需常態性大規模向海內外設備大廠採購高階運算等級之高速交易 [[伺服器]] (Servers) 叢集與核心超低延遲光纖 [[網通]] 設備；同時嚴重仰賴各跨國直連海纜專線基礎設施網供應商及即時報價串流端。
